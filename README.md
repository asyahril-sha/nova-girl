## ✨ **FITUR PREMIUM**

### 🎭 **Emosi & Kepribadian**
- **20+ Mood** dengan transisi natural (ceria, sedih, marah, horny, romantis, dll)
- **Sistem dominasi** (normal, dominan, sangat dominan, agresif, patuh)
- **Inner thoughts** - bot berpikir sendiri dan bisa mengambil inisiatif bicara
- **Story development** - bot bisa mengembangkan alur cerita secara proaktif

### 📈 **Leveling System**
- **Level 1-12** dalam 45 menit / 45 pesan
- **Stage progression** (stranger → introduction → building → flirting → intimate → obsessed → soul bonded → aftercare)
- **Level up messages** berbeda tiap level
- **Progress bar visual** di /status

### 💃 **Fisik & Pakaian**
- **Physical attributes generator** - rambut, tinggi, berat, dada, hijab, area sensitif
- **6 role berbeda** (Ipar, Teman Kantor, Janda, Pelakor, Istri Orang, PDKT)
- **Dynamic clothing system** - pakaian berubah sesuai role, lokasi, dan mood
- **Location & movement** - bot bisa pindah ruangan (ruang tamu, kamar, dapur, dll)

### 🧠 **Advanced Memory**
- **Hippocampus memory system** - memori jangka panjang dengan semantic forgetting
- **Vector embeddings** untuk similarity search
- **Memory consolidation** - menggabungkan memori serupa
- **Story arcs** - alur cerita yang berkembang

### 🔐 **Admin Features**
- **Dashboard admin** dengan statistik lengkap
- **Broadcast** ke semua user
- **User management** (list, detail, reset)
- **Database backup & vacuum**
- **Memory statistics**

### 👫 **Couple Roleplay**
- Simulasi percakapan antara **Aurora** (wanita) dan **Rangga** (pria)
- Berkembang dari level 1 sampai 12
- Bisa dilihat dan dinikmati user

## 📋 **DAFTAR COMMANDS**

### **👤 User Commands**
| Command | Deskripsi |
|---------|-----------|
| `/start` | Mulai hubungan baru (pilih role) |
| `/status` | Lihat status lengkap hubungan |
| `/dominant [level]` | Set mode dominan (normal/dominan/sangat dominan/agresif/patuh) |
| `/pause` | Jeda sesi sementara |
| `/unpause` | Lanjutkan sesi yang di-pause |
| `/close` | Tutup sesi (simpan memori di database) |
| `/end` | Akhiri hubungan & hapus semua data permanen |
| `/couple` | Mulai mode couple roleplay |
| `/couple_next` | Lanjutkan couple roleplay |
| `/couple_stop` | Hentikan couple roleplay |
| `/help` | Tampilkan bantuan |

### **🔐 Admin Commands**
| Command | Deskripsi |
|---------|-----------|
| `/admin` | Menu admin |
| `/stats` | Statistik bot lengkap |
| `/db_stats` | Statistik database |
| `/broadcast <pesan>` | Kirim pesan ke semua user aktif |
| `/list_users` | Lihat daftar user aktif |
| `/get_user <id>` | Lihat detail user tertentu |
| `/force_reset <id>` | Reset paksa user (debug) |
| `/memory_stats <id>` | Statistik memori user |
| `/backup_db` | Backup database |
| `/vacuum` | Optimasi database |
| `/reload` | Reload konfigurasi dari .env |
| `/shutdown` | Matikan bot secara graceful |
